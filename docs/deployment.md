# Deployment

## Status

Deployment is intentionally deferred until the infrastructure, persistence, and runtime integrations are real.

## Planned Phase 9 Scope

- production Dockerfiles for `web`, `api`, and `worker`
- reverse proxy and public routing
- environment and secret mapping
- deployment target selection
- release checklist and operational verification

## Interim Constraint

The current Phase 1 scaffold is a development foundation, not a deployable product build.
